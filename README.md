Samuel Andrade Resende — Portfólio
Sobre o Projeto

Este é meu portfólio pessoal desenvolvido com o objetivo de apresentar minha trajetória, habilidades técnicas e projetos na área de tecnologia.

O projeto foi construído com foco em design moderno, organização de código, interatividade e experiência do usuário, utilizando HTML, CSS e JavaScript.

O site está publicado online via Vercel:

https://curriculo-three-beta.vercel.app/

Sobre Mim

Sou estudante de Sistemas de Informação e estou em constante evolução na área de tecnologia. Iniciei minha trajetória profissional ainda jovem, atuando na padaria da minha família, onde desenvolvi responsabilidade, organização e visão prática de negócios.

Em 2024, me mudei para Belo Horizonte para iniciar a graduação e direcionar minha carreira para tecnologia. Desde então, venho desenvolvendo projetos práticos envolvendo APIs, modelagem de banco de dados, estruturação de sistemas e lógica de programação.

Tenho interesse em arquitetura de sistemas, organização de código e desenvolvimento back-end, buscando sempre entender não apenas como implementar soluções, mas como estruturá-las de forma escalável e eficiente.
